## 📌 Project Overview
In this project, I learned about digital literacy and how important it is in today’s world. The project helped me understand how to use technology safely, create useful content, and build my online presence. I also got practical experience with different digital tools and platforms.

---

## 📁 Tasks Included

### 🔹 Task 1: Presentation
In this task, I created an infographic using Canva. It helped me understand how to present information in a visual and simple way. I included points like online safety, digital tools, and responsible use of technology.

### 🔹 Task 2: Digital Portfolio
I created and updated my profiles on GitHub, LinkedIn, and Kaggle. This helped me understand how to build a professional online presence and showcase my skills.

### 🔹 Task 3: Platforms
In this task, I explored different digital platforms and learned how they are used for learning, collaboration, and sharing information.

### 🔹 Task 4: Email Etiquette
I learned how to write professional emails and understood the importance of proper communication. I also learned the do’s and don’ts of using social media.

### 🔹 Task 5: Cybercrime Awareness
In this task, I learned about cybercrime, especially phishing attacks. I also understood how to stay safe online and protect personal information.

### 🔹 Task 6: Project Report
Finally, I compiled all the tasks into a single report, which helped me revise everything I learned during this project.
